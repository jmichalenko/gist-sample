Syntax

This was one of the first projects done in the c programing language.

The purpose was to learn: What syntax is; When to use parentheses and double quotes ; Why we use curly braces and semicolons and The syntax needed to create a program in C

Variables

A variable is a container that holds numbers, words, or other types of data to use in our program.

The purpose of this lab was to learn: How to create and naming a variable; How to assign a value to a variable and How to get user input for a variable

Data Types

A data type, in programming, is a classification that specifies which type of value a variable can hold.

The purpose of this lab was to learn: About common data types; Problems that can arise with ints and floats and Data input functions for different data types
